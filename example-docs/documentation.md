## Sum

Adds two numbers together.

### Props

| Name  |   Type   | Description        |
| :---- | :------: | :----------------- |
| **a** | `number` | The first number.  |
| **b** | `number` | The second number. |

### Code Example

```js
const sum = (a, b) => a + b;
```
